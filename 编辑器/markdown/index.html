如何使用 markdown 编辑器

本文字数： 1.4k 阅读时长 ≈ 1 分钟

什么是 markdown

Markdown是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与HTML混编，可导出 HTML、PDF 以及本身的 .md 格式的文件。因简洁、高效、易读、易写，Markdown被大量使用，如Github、Wikipedia等网站，如各大博客平台：WordPress、Drupal、简书等。

官方解释，官网地址：Markdown

Markdown 编辑器推荐 –Typora

官网地址: typora

什么是Typora

Typora 是一款好用的极简且免费的 markdown 编辑器和阅读器

为什么选择 Typora

  1. 开源免费

  2. 跨平台

    目前仅有 windows 与 Linux

  3. 支持图床自动上传

  4. 实时渲染

Markdown 如何使用

1、结构类操作

(1)多级标题

  1. 通过#来实现

    1. # 一级标题
    2. ## 二级标题
    3. 最多支持六级标题

  2. 通过快捷键实现

    1. Ctrl+1~6 添加标题
    2. Ctrl+0消除

(2)列表语法

有序列表

  1. 使用方法(例)

    1. 打开冰箱
      1. 插电
      2. 开机
      3. 开门
    2. 把大象放进去
    3. 关闭冰箱

  2. 编号后一定要有空格

  3. 行末回车，序号会自动添加。行末↓键，序号不会添加。

  4. 删除/增加某一行内容，序号会自动调整。

  5. 降级快捷键Tab,升级快捷键Shift+Tab

快捷键：Ctrl + Shift + [

####无序列表

  1. 例：

  • 青菜

  • 水果

    1. 苹果
    2. 梨子
      1. 菠萝
        • 国产菠萝
        • 进口菠萝
    • 肉
  1. 代码： ** 青菜* 或 - 青菜
  2. 注意事项
    • *或-后，一定要有空格！
    • 有序列表和无序列表可以交错使用。

快捷键: Ctrl + Shift + ]

(3)表格

学号姓名性别年龄
001张三男20
002李四女21

创建方式: |学号|姓名|性别|->回车。

添加一行，只需要按下Ctrl + 回车

插入表格快捷键Ctrl + t

2、 注释引用类

(1)任务列表

  1. 例：
      • 青菜
      • 水果
      • 肉
  2. 源代码： - [ ] 青菜
  3. 注意事项:
    1. - [ ] 后都一定要有空格
    2. 无快捷键可用过鼠标操作：选中文字->段落->任务列表

(2) 强调语法

  1. 例

    1. 倾斜
    2. 加粗
    3. 加粗倾斜

  2. 源码:

    1. *倾斜*
    2. **加粗**
    3. ***加粗倾斜***

    使用*进行操作

(3)引用语法

  1. 例

    引用，注释

  2. 源码

    > 引用注释

    > 后一定要有回车

(4)转义字符

通过在markdown特殊字符钱加入\ 用于格式化 Markdown 中的特殊标记符号

例如

> 这是一个大于号

去除\后，markdown 会对其进行渲染

这是一个大于号

3、代码类相关操作

(1)行内代码

创建方式： 行内代码–两个反引号(`)扩起来

快捷键Ctrl + Shift + `

(2)代码块

第一个Java代码

创建方式*```java* 或 ~~~java

快捷键Ctrl + Shift + k

  1. 诺念
  2. 代码: [诺念](https://nuonianpass.cn)
  3. []中输入你想要的文字，后面圆括号后为网址

  2. 源码
