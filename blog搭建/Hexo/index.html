Windows

官网 www.nodejs.org 下载LTS版本，安装直接下一步即可

Linux

安装方式同 Git ，只需将相应命令中的git换为nodejs npm即可

验证方式同 Windows

npm换源

考虑国内的网络环境，npm下载速度过慢，安装 cnpm 并使用淘宝镜像源以提升下载速度，安装后将使用 cnpm 命令替换 npm 命令，如感觉自己网络状况良好 or 可魔法上网 可以自行忽略此步骤，且后续操作继续使用 npm 命令
